# Mirai Projects

Welcome to the `mirai` repository!

## Contents

- **[Identity Echo Interface](./Identity-Echo-Interface/)**: A polished Streamlit web application that lets users enter their name and message, then transmit the information with a modern and friendly interface.

## Getting Started

To run the Identity Echo Interface:

```bash
cd Identity-Echo-Interface
pip install -r requirements.txt
streamlit run app.py
```
