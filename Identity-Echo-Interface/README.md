# 📡 Identity Echo Interface

A polished Streamlit web application that lets users enter their name and message, then transmit the information with a modern and friendly interface.

![Screenshot Placeholder](https://via.placeholder.com/800x450?text=Identity+Echo+Interface)

## ✨ Features

- Clean and modern Streamlit UI
- Name and message input fields
- Validation for empty fields
- Success, warning, and error feedback
- Estimated AI token usage display
- Sidebar project information
- Footer developer credit
- Responsive column-based layout

## 🛠 Installation

1. Clone or open the project folder.
2. Install the dependencies:

```bash
pip install -r requirements.txt
```

## ▶️ How to Run

Start the app with:

```bash
streamlit run app.py
```
